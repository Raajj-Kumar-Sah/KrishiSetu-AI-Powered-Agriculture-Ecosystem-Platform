## 📖 Overview

**KrishiSetu** is a modern **AI-powered agriculture ecosystem** designed to digitally connect **farmers, buyers, equipment providers, agro stores, delivery partners, and administrators** on one unified platform.

The platform enables farmers to sell crops directly, rent agricultural machinery, detect crop diseases using AI, receive intelligent recommendations, and track deliveries in real time.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/KrishiSetu.git

cd KrishiSetu
```

---

## Backend

```bash
cd backend

npm install

npm run dev
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
